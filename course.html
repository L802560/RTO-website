Courses description

1.ESSENTIAL DEVELOPER TOOLS
{ Terminal and UNIX. }

Welcome to Rithm's Terminal and UNIX course. In order to be a succesful developer, it's essential that you be comfortable in the terminal. In this course, we'll walk you through the basics of navigating through files and folders in the terminal. We'll also dicuss in detail how to see and set permissions, chain commands together using piping, and create environment variables. By the end of the course you'll also have gained some exposure to more advanced terminal commands, and will be able to set up your own remote server through Amazon Web Services!
{ Git and GitHub.}

Welcome to Rithm's course on Git and GitHub! An understanding of these technologies is essential when you're working with other developers. In this course, we'll talk about the importance of using a version control system, and we'll introduce the basics of incorporating Git into your workflow. You'll learn how to create branches to work on your own features, and how to resolve conflicts in your code. By the end of the course, you'll also be confident in your ability to use GitHub to share your code, and you'll know a few advanced Git tricks as well.
2.HTML + CSS FUNDAMENTALS
{ HTML and CSS Basics. }

Welcome to Rithm's introduction to HTML and CSS! It's impossible to make a good-looking website without these technologies. In this course, you'll learn about the basics of HTML, including semantic HTML with HTML5. You'll also learn about some of the most important CSS properties and rules that govern the look of a web page.
{ Intermediate CSS and Twitter Bootstrap. }

Welcome to Rithm's intermediate CSS and Bootstrap course! Once you've mastered the basics of CSS, this course will teach you about some more advanced CSS features, including transitions and animations. You'll also learn the basics of responsive design, which you can use to help make your site look good on a variety of screen sizes. Next, you'll learn about Twitter Bootstrap, a free library that will help you create good-looking sites as quickly as possible. Finally, you'll learn how to share your creations with the rest of the world by deploying your site to the world wide web!
3.JAVASCRIPT FUNDAMENTALS
{ JavaScript Fundamentals. }

Welcome to Rithm's free JavaScript fundamentals prep course! The material here will guide you through the basics of JavaScript. You'll learn how to store data in variables, write code that runs only if certain conditions are met, and write functions. By the end of the course, you'll be able to build simple games entirely in JavaScript!
{ Intermediate JavaScript Part I. }

You’ve made it to the first part of Rithm's free Intermediate JavaScript course! You’re ready to learn how to debug like a professional, access and create values in more complex data structures, dive deep into callbacks and closures, and learn all about event-driven programming with the DOM. You'll also learn about some more advanced built-in methods on arrays. By the end of the course, you'll be able to build interactive browser based games and applications with JavaScript! If you want to go back and review some fundamentals, head over to our JavaScript Fundamentals course.
{ Intermediate JavaScript Part II. }

You’ve made it to the second part of Rithm's free Intermediate JavaScript course! You’re ready to learn how to use jQuery and write clean jQuery code, understand in depth how the web works, and use the XHR object to make AJAX calls. By the end of the course you'll be able to retrieve information from external APIs like Google Maps to build even more sophisticated applications! If you want to go back and review the previous chapters, head over to our Intermediate JavaScript Part I.
4.ADVANCED JAVASCRIPT + COMPUTER SCIENCE IN JAVASCRIPT
{ Advanced JavaScript Part I. }

You’ve made it to Rithm's free Advanced JavaScript course! You’re ready to learn more difficult concepts in JavaScript like the keyword this, call/apply/bind, object oriented programming, and more. We'll revisit concepts from the three previous JavaScript courses, so be sure you have a solid understanding of that material before you continue with this course.
{ Advanced JavaScript Part II. }

You’ve made it to the second part of Rithm's free Advanced JavaScript course! You’re ready to learn more advanced concepts in JavaScript like canvas, regular expressions, promises, generators, async functions, ES2015 object oriented programming, regular expressions, managing asynchronous code, and more. We'll revisit concepts from the four previous JavaScript courses, so be sure you have a solid understanding of that material before you continue with this course.
{ Computer Science Fundamentals with JavaScript. }

Welcome to Rithm's free Computer Science fundamentals course! The material here will cover some basic algorithms and data structures in computer science. You'll use JavaScript to implement different searching and sorting algorithms, and you'll learn about linked lists, trees, and more. You'll also learn how to compare the efficiency of different algorithms using big O notation. In order to be successful in this course, you will not only need to have prior experience writing JavaScript, but also with Git and GitHub so that you can fork and clone repositories and run tests locally.
